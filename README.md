# 🎬 Netflix Content Strategy Analysis (Python EDA)

## 📌 What this project is about

Analyzed Netflix dataset (~8000 titles) to understand:

- how content is distributed  
- what kind of content Netflix focuses on  
- which regions contribute the most  
- how content has evolved over time  

---

## ⚙️ Tools Used

- Python  
- Pandas  
- Matplotlib, Seaborn  

---

## 🧠 Key Questions Explored

- Movies vs TV Shows — what dominates?  
- Which countries produce the most content?  
- What genres are most common?  
- Has Netflix increased content over time?  
- Are newer titles longer or shorter?  

---

## 📊 Key Findings

- 📌 **Movies dominate the platform**  
  → ~70% of total content is movies  

- 📌 **Content growth accelerated after 2015**  
  → strong increase in releases over recent years  

- 📌 **US leads content production**  
  → followed by India and the UK  

- 📌 **Drama & International content dominate genres**  
  → consistent across both Movies and TV Shows  

- 📌 **No strong relationship between release year and duration**  
  → correlation ≈ 0.1 (very weak)  
  → content length has remained fairly stable  

- 📌 **TV Shows do not necessarily increase in seasons over time**  
  → weak correlation (~0.11)
